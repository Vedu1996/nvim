# My NeoVim Config

Inspired by [typecraft](https://www.youtube.com/watch?v=zHTeCSVAFNY&list=PLsz00TDipIffreIaUNk64KxTIkQaGguqn&index=1)

Uses:

- Lazy Package Manager
- Treesitter
- Mason
- Telescope
- Lualine
- LSP Config
- StatusCol
- Conform

Works well for:

- JavaScript
- Typescript
- Lua
- Markdown

Language Servers / Formatters used:

- ts_ls
- eslint_d
- prettier
